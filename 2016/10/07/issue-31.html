Issue #31

A round up of what’s happening in the wonderful world of the Elm programming language.

Reading

Decoding JSON With Dynamic Keys

Brian Hicks deals with the thorny problem of JSON dynamic keys.

Building a single page application with Elm

Dave Thomas builds a single page application with Elm and delivers an experience report.

What does it mean when code is “easy to reason about”?

Preethi Kasireddy brings clarity to the much abused phrase in a well-disguised advertisement for Elm.

So You Want to be a Functional Programmer (Part 5)

Charles Scalfani continues his detailed introduction to Functional Programming.

Elm editor plugin comparison

A spreadsheet with a comparison of Elm features and plugins in popular text-editors / IDEs.

Atom packages: elm-instant 0.4.5, elmjutsu 2.8.2
Vim plugin: updates to elm-vim 

Watching & Listening

Elm: Delightful Web Development

Amir Barylko delivered this introductory talk at the Winnipeg .NET User Group in June.

Talking about Elm

Marcus Griep’s talk from LambdaConf in May.

Elm in Purescript

Ryan Rempel talks about his library Purescript-Elm focusing on some of the design choices Elm has made to appeal to beginners.

Live coding - fixing a bug in my elm narrative engine

Jeff Schomay thinks about making impossible states impossible in this live stream recording (see also his Elm Conf talk).

ElmLive - Random Maps

Aaron VonderHaar codes with fuzz tests (property tests), making a random map generator using test-driven development. 

ElmLive - RPG Platform

Greg Ziegan grabs some free sprites, to make his RPG look more like a real game, and crafts a tile builder.

Elm-Test 2.1

An overview of significant API changes since Elmseed’s first screencast on elm-test with Erik Person. Also json-decode-extra.

Elm Revisited

The Web Platform Podcast speaks with Richard Feldman and Conner Ruhl in Episode 108.

Code

SvgBob

SvgBob converts ascii scribbles into an SVG with Elm, by Ivan Ceras (code). Amazing.

Visualization of Okasaki's Purely Functional Data Structures

Michael Anderson has started a project to build visualizations from Okasaki’s popular book in Elm (code).

L-System Playground

An L-system implementation with turtle graphics for Elm by Alexander Klink (code).

Elmalytics

Michael Bylstra crunches some numbers from the GitHub API to plot Elm’s rise (Elm frontend, Python backend).

elm-run

Task-based bindings for running Elm in the console from Kofi Gumbs.

elm-time 1.0.1

Bogdan Popa releases a pure Elm date and time library.

elm-style-animation 3.1.0

Matthew Griffith releases version 3.1.0 of the style animation library for Elm (see his Elm Conf talk).

Add a scroll progress indicator from Chris Buttery (code).

Connect your game pad to the browser with elm-gamepad from Francesco Orsenigo.
Gizra’s Yeoman generator for Elm (now 1.5.0) has been adopted by David Hernández.
Build a game in an exercise from Elmoin’s October Meetup.
