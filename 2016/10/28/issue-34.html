Issue #34

A round up of what’s happening in the wonderful world of the Elm programming language.

Reading

Language-Level Reactivity with Elm

Richard Feldman shows how Elm’s fundamental embrace of reactivity offers a better experience.

Pagination

Roman Zolotarev uses pagination as a simple problem to compare JavaScript and Elm solutions.

Making elm-test More Modular

Bill Peregoy shares some Elm testing tips. See also his article on Testing JSON decoders in Elm.

Decoding dynamically shaped JSON in Elm

Jacky offers a solution to what can appear a thorny problem.

A Generic Handler For AWS SDK Operations in Elm

Matt Pfefferle uses the AWS SDK for JavaScript to query DynamoDB directly from the browser.

Building a Facebook-like Comment Box

A React tutorial, in Elm by Rudolf Olah.

Not for the faint of heart: Optical Swordplay With Components:
Using Lenses & Monads to Refract the Elm Update by Kyle J. Kress.

Elm & Express receive some attention from Andy Peterson
A Simple Client-Server Implementation and Nick Balestra Full-Stack Web Development.

David Nimmo was wrong to dismiss Elm, and he thinks you probably are too (although based on your reading habits, you haven’t dismissed it, have you?); Chris Buttery rounds up his robust experiences with Elm listing a lot of his code (some of which has been featured here); and James Kolce beautifully typesets this getting started post.

Watching & Listening

API Design Session: Elm Narrative Engine

Evan Czaplicki sits down with Jeff Schomay for an API design session to share some wisdom on how folks can improve their own designs. 

Elm Live: implementing "undo" in my narrative engine

Jeff Schomay does it in under 15 minutes!

Functions in Elm

An Egghead screencast covering stateless functions, function composition, anonymous functions, Currying, and more with Ronn Ross.

Elm for JavaScripteers

Mike Anderson introduces Elm to a JavaScript audience in this talk.

Elm Tutorial Part 4: Analog Clock App

In this tutorial, Tensor looks at an Analog clock app and talks about Elm subscriptions.

Making a Multiplayer Memory Game with Elixir and Elm

Max Gorin’s Lightning talk from EuRoKu in Sofia.

Building a Pomodoro App

Arron Mabrey and Tony Winn pair program a Pomodoro App live using Phoenix Channels and an Elm front end.

Creating Libraries for Elm: Lessons Learned

Slide deck from Ramón Soto Mathiesen’s talk at Elm Copenhagen.

JSON Street

The Elm Town podcast returns with episode 3. Host Murphy Randle talks JSON with Brian Hicks.

Code

Elm Twitter client for Chrome

Elm Twitter client for Chrome by Marcelo Lazaroni.

Benchmarking Elm Functions

Patrick Oyarzun publishes a package to help benchmark your Elm code.

formatting 4.0.5

Kris Jenkins releases the latest version of his string formatting library (like printf, with interpolation).

Netlify’s playground

An application built in Elm to test redirect rules for Netlify’s static hosting service (code).

Loops With Friends

Start making music collaboratively with this real-time Elixir and Elm app by Jeff Cole (code).

Bowling Game Kata

Uncle Bob’s Bowling Game Code Kata in Elm from Pierre Beitz (also the Gilded Rose Code Kata)

reading-time

Medium-esque reading stats in elm by Chris Buttery (also is-online).

type-safe-json-decoder

Elm inspires this type-safe JSON decoder for TypeScript by Wesley Merkel.

Mastermind game written in Elm

Mastermind game written in Elm by Damien Le Berrigaud including tests.
