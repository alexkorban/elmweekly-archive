Issue #37

A round up of what’s happening in the wonderful world of the Elm programming language.

News

Elm Beyond the Basics

James Moore’s popular video course has been updated for 0.18, Learn How to Build Web Apps With Elm. Also updated, his free course: Elm for Beginners.

Elm in Action: Chapter 4, Talking to Servers

An update from Richard Feldman on his Elm book which sees chapter 4 released, and all content updated for version 0.18.

Reading

Property Based Testing And Better Modelling

Lucas Reis dives into data modelling and property testing in part 4 of his Learning Elm series.

Building a memory game in Elm. Step by step, from scratch.

A comprehensive walkthrough of building a memory game in Elm with Marco Sampellegrini.

Farewell to Backticks, Hello Pipeline!

KKruups writes about a new feature in 0.18 and delves into the why. Also How the elm JSON API, Elm-Decode-Pipeline and Json.Decode API Work.

Adventures in Elm

A festive contribution from Florian Verdonck who builds “Secret Santa” in Elm. 

Runtime Expectations as Package Constraint Dimensions

Luke Westby thinks about what a package manager might look like if we applied Elm’s sensible package management to the runtime platform.

Functional Sets, Part 3: Balancing

Brian Hicks performs a delicate balancing act, ensuring we’re not going too far to the left or right.

Making Elm code more compact and more readable

Wouter In t Velt uses extractions for friendlier code. 

Watching & Listening

Elm and React in production

Richard Feldman takes a deep dive into Elm and React from a production perspective—what's familiar, what's different, and how they can work together—in this talk from October’s ReactiveConf.

Why Elm?

Luke Westby makes a good case for Elm in this SitePoint screencast.

Web Components Introduction

A quick primer on using Web Components with Elm from Josh Adams in this free Daily Drip screencast.

A Little Story About the Elm Debugger

Mike and Nicole Clark show off the new 0.18 debugger in this short screencast.

Demos of a front-end debugger for a NES emulator written in Elm by W. Brian Gourlie and a React native app with an Elm brain by Anthony Naddeo.

Interested in watching Elm Live streaming, live? or want to join the fun? Check out this calendar.

Code

Silver Magpie

The cleanest Twitter client for Chrome, an open-source Chrome extension in Elm by Marcelo Lazaroni (code).

elm-static-html

Noah Hall allows you to turn your static Elm into static HTML pages (i.e., no JavaScript, just HTML). 

Elm Visuals

Damien Tardy-Panis plays with the Elm tangram, putting together a collection of images and animations (code).

elm-dynamodb

An Elm interface to Amazon’s DynamoDB by Bill St. Clair along with comprehensive documentation.

elm-draggable-tabs

Some sweet drag ’n’ drop tabs (experimental) from Greg Ziegan (code).

elm-calendar-react-storybook

Simple demo of how ReactStoryBook can be used with an Elm App by Kalutheo.

vim-tagbar-ctags-elm

Support for Elm in Vim Tagbar with Universal Ctags by Mark Skipper.
