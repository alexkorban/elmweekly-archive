Issue #23

A round up of what’s happening in the wonderful world of the Elm programming language.

News

Elm In Action

An early access edition of Richard Feldman’s book, Elm in Action, is now available for purchase. It features a completed draft of the first two chapters.

Elm-lang For Beginners

James Moore has released a new video course on Elm for beginners. Available for free for a limited time.

Reading

Maybe the Cookies API should not exist

Evan Czaplicki takes us through the thought process that led to not wrapping the JavaScript Cookie API with Elm. 

Child-Parent Communication, continued

Folkert de Vries rounds up a number of articles about inter-component communication.

A hierarchical data transformation DSL in 3 lines of Elm

James MacAuley writes a story about composing trees of computation in Elm.

First impressions of Elm

Esteban Manchado Velázquez shares his thoughts on Elm along with a demo and code of his first application. 

Watching & Listening

Creating a Fun Game with Elm

Video of a talk by Andrey Kuzmin and Kolja Wilcke at Curry On, Rome last week. The Slides (Code) feature one of the best Elm architecture diagrams yet!

Destructuring Screencast

Erik Person releases a new Elmseeds screencast covering destructuring.

Designing with Passion

Evan Czaplicki features on a panel of language designers with Joe Armstrong (Erlang), John Hughes (Haskell, QuickCheck), with author and language enthusiast Dave Thomas and chaired by Bruce Tate.

Decode a List of Numbers From a JSON String in Elm

Murphy Randle explains the basics of JSON decoding in this Egghead.io screencast.

Code

Extending TEA

Matias Klemola and Ossi Hanhinen have put together an example of extending The Elm Architecture stemming from real production usage in a Futurice project.

Elm Charts: A pure Elm chart library

An alpha version of a pure Elm chart library 📈 by CT Wu.

Battery Status

Keep track of the user's battery power status. Code from Luke Westby.

Example repository from Elm TLV 3rd meetup

From a simple counter to (almost) a full SPA example, in small steps. A tutorial from Elm TLV meetup by Amitai Burstein.

Tic tac toe game in Elm lang

Inspired by an entry in last week’s issue, Daniel Garcia has written his own version of Tic Tac Toe. He added it might be useful if an Elm Expert had a go, to compare approaches.
