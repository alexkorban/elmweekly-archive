Issue #25

A round up of what’s happening in the wonderful world of the Elm programming language.

News

elm-mdl 7.0.0 Released

The Elm port of Google’s Material Design Lite is now feature complete, supporting all components, in version 7.0.0. Read this announcement post for all the details. 

Reading

Values, Pipes, and Arrows

Brian Hicks peers into pipes in Elm.

Introduction to using Material Design in Elm

Daily Drip’s Josh Adams writes an introductory tutorial to elm-mdl hot on the heels of its recent release.

Creating Your First Elm App: From Authentication to Calling an API

A two part series from Auth0’s Kim Maida involving HTTP and authentication with JSON Web Tokens. (Part 2)

First weeks in Elm

AJ Robertson delivers his take on Elm’s strengths and suitability.

Realtime Elm Revisited with Elm 0.17

Jack Franklin revisits his Elm Pusher integration for 0.17 discovering less code and less complexity.

Sending optional data in Elm with Maybe, Result and Task

Tolga Paksoy explores core Elm functions when dealing with optional data in (HTTP) tasks on the Paramander blog.

Playing with WebSockets in Haskell and Elm

Mats Rietdijk builds a small type-safe WebSocket server and client with Haskell and Elm on the Paramander blog.

Design of Large Elm apps

Richard Feldman offers perspectives from NoRedInk’s codebase in this mailing list thread about application architecture.

Watching & Listening

ElmLive - Drag'n'Drop

More live streaming from Aaron Vonderhaar featuring drag'n'drop in Elm in building a photo layout editor. He also touches on JSON decoding and HTTP requests.

elm-mdl Introduction

An introduction to using Google's Material Design language in Elm, with elm-mdl from Josh Adams on Daily Drip. 

elm-mdl: Textfields and Menus

A super brief introduction to text fields and menus in elm-mdl from Josh Adams on Daily Drip.

Auto-Compilation

Erik Person looks at using Grunt to auto-compile Elm code in this Elm Seeds screencast.

Elm Pair Programming: Colluder

Josh Adams and Luke Westby record their pairing session on Colluder, an audio toy in Elm.

Code

elm-test 2.0.0

elm-test has hit the big 2.0.0. The new release, spearheaded by Richard Feldman and Max Goldstein, offers an improved API and the addition of Fuzz testing.

Curator’s Corner

Shape of errors to come in Rust

Not Elm per se, but great to see Elm’s influence spreading further afield in this article describing Elm as the inspiration for Rust’s new error messages.
