Issue #26

A round up of what’s happening in the wonderful world of the Elm programming language.

News

Elm in Action tops bestseller chart

The early access version of Richard Feldman’s Elm in Action hits the top of publisher Manning’s bestseller list.

Elm Training Berlin

Zachary Kessin is holding a three day Elm training event in Berlin on 14th–16th November.

Reading

Welding Functional Pipes

Brian Hicks follows last week’s post on pipes with techniques for function composition.

toast.al / Getting Ourselves Organized

Kyle J. Kress looks at file and folder Structure for an Elm Single Page Application.

SPAs in Elm: Part 4 – Side Effects

Kevin Greene continues his tutorial on Single Page Applications in Elm by examining side-effects.

Interacting with a DOM element using Elm

Vincent Jousse uses the <audio> tag to explain how you might work with DOM elements in Elm.

Type Safe JSON Decoding in Elm

Charles Scalfani sheds light on type safe JSON decoding.

Watching & Listening

API Design Session – Focus and Scrolling

Evan Czaplicki pairs with Richard Feldman in the second recorded API Design Session, with a focus on the DOM.

Solitaire game #1

Martin Janiczek streams the building of a solitaire game (code).

Elm – Frontend with Guarantees

Amitai Burstein’s talk from the You Gotta Love Frontend conference in Tel Aviv at the end of June.

Livestream/Recording setup with OBS Studio

Aaron VonderHaar shares his streaming setup for anyone interested in doing something similar.

Code

Cognitive Control Training

Train your brain with cognitive control training written in Elm by Francesco Orsenigo (code).

elm-spinner

Damien Klinnert has put together a highly configurable, efficiently rendered spinner component. 

Practical examples in Elm

A collection of examples with advanced techniques for real-world Elm apps from Eduard Kyvenko. (See also his Sierpinski Fractal)

Lichter Aus!

The “Lights Out!” puzzle implemented by Carsten König.

Lines and Points Effect

A small drawing / animation using SVG and Elm Ivan Acosta-Rubio.

Multiplayer Tic-Tac-Toe

A multiplayer game of Tic-Tac-Toe using Node.js, WebSockets and Elm assembled by David Lucia.
