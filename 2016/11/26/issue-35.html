Issue #36

A round up of what’s happening in the wonderful world of the Elm programming language.

News

Elm 0.18 Released

In “The Perfect Bug Report” Evan Czaplicki outlines a superlative debugging experience with this new version of Elm.
(Upgrade Guide | Better Errors)

Announcing New Course for Elm 0.18

The Pragmatic Studio duo Mike and Nicole Clark have brought out the first videos of a new version of their popular Elm course updated for 0.18.

Elm Europe 2017

Early Bird tickets are available for next year’s first European Elm Conference on 8–9 June in Paris, France. They are also accepting talk proposals. 

Start Using Elm to Build Web Applications

Murphy Randle has released a free Elm course on EggHead.io with 30 mins of introductory content (version 0.17).

Reading

Upgrading to Elm 0.18

Bill Peregoy describes his manual upgrade to 0.18, also SPAs with routing, Polling, Subscriptions and Time, and Tasks in 0.18.

CRUD SPA with Haskell and Elm: Elm 0.18 Upgrade

Magnus Rundberget upgrades his Haskell + Elm app to 0.18 in part 6 of his series.

Building custom DOM event handlers in Elm

A step-by-step guide to implementing a custom `onChange` event in Elm 0.18 by Joël Quenneville.

Build an Instagram Clone in Elm: InstaElm Part 1

Rudolf Olah clones Instagram with Elm on the CodeMentor blog (0.17).

Elm Tools and Resources

A list of tools and resources Dennis Reimann found valuable when working with Elm, containing useful tools for day to day work and links to deepen your Elm-fu. Also Elm Atom Editor and Elm Type Annotations.

Happy Little Trees: Decoding Recursive Data Structures in Elm

Brian Hicks looks at Elm trees and JSON decoding as well as starting a new series on Data Structures in Elm: Functional Sets Construction and Rotation.

On valid models and data structures

Marco Perone writes about how he changed his approach to data modelling to simplify his code.

Watching & Listening

0.18 in Elm Town Podcast

The fifth episode comes with a focus on Elm 0.18 featuring Murphy Randle, Noah Hall, Aaron VonderHaar and Luke Westby.

Upgrade to Elm 0.18

Erik Person screencasts an upgrade to 0.18 and also covers advanced tasks.

Functional patterns for creating elegant applications in Elm

José Lorenzo Rodríguez’s Elm talk from the Øredev conference in Malmö, Sweden earlier this month (Slides).

Building a Graphical IDE

Claudia Doppioslash gave a talk at CodeMesh in London on building a Graphical IDE in Elm/Purescript for an Embedded Language that Compiles to the Erlang VM.

Types As Design Tools

Kris Jenkins’ talk from CodeMesh on using types as design tools (containing lots of helpful Elm error messages). 

Codevember 2016

Aaron VonderHaar’s playlist from his Codevember, daily Elm for the month of November, with some cool experiments in WebGL.

See also Jeff Schomay’s Codevember contribution with Elm Narrative Engine.

Elm Live Experiment: Worker Renderer

Part 1 of live coding an experiment to run an Elm program's Model and Update in a Web Worker with Luke Westby (Part 2).

Elm + Electron QA Time Travel

Charlie Koster (and team) demo an Elm + Electron QA, the time travel features include working with HTTP requests, Electron API invocations, obfuscating sensitive data that shouldn't be recorded, and breakpointing a playback file with Chrome DevTools.

Code

elm-doc-test

Turn your documentation into tests! Christoph Hermann provides a package to enable comments written to a specific format to be converted to Elm test code (similar to Python’s doctest).

Elm Hangman

Runar Furenes builds a hangman game with a type-safe frontend (code).

ElmDocs

An experimental Elm Documentation browser (à la DevDocs) written in Elm by JackySee (code). I like the cross package function search.

Elm IndexedDB Example

Persist Elm data structures to IndexedDB using ports and LocalForage by Daniel Walker.
